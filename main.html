Default parameters

The default value for function parameters is undefined. This can be achieved with the help of default function parameters.

      
        function fullName(firstName, lastName ='John', fullName = firstName + ' ' + lastName ) {
          return fullName;
        }
        fullName('Daliya'); // return Daliya John

Template literals (Template strings)

Template literals allow embedded expressions, multi-line strings and string interpolation.

      
        let myName = 'Daliya';
        let myRole = 'Software Developer';
        console.log(`My name is ${myName} and I am a ${myRole}.`); // My name is John and I am a Software Developer

Tagged Templates

Parse template literals with a function.

      
        let myName = 'Daliya';
        let myRole = 'Software Developer';
        function myTag(array, name, role){
          console.log(array);
          let str0 = array[0]; 
          let str1 = array[1]; 
          return `${str0}${name}${str1}${role}`;
        }
        myTag`My name is ${myName} and I am ${myRole}`;

Destructuring assignment

The destructuring assignment allows reading values from an array or properties from an object, into distinct variables.

Destructuring an array
      
        let myName2, myRole2;
        let array = ['John', 'Software Developer'];
        [myName2, myRole2] = array; 
        console.log(myName2, myRole2);
Destructuring an object
      
        let myNameTest, myRoleTest;
        let object = {
          myNameTest:'Daliya', 
          myRoleTest:'Developer'
        };
        ({myNameTest, myRoleTest}=object);

Const

A const statement does not allow reassignment or redeclaration of the variable. The value is mutable.

      
        const myName; //throws error as const needs to be initialized
        const myName='John';
        const myName='Doe'; //throws  error as const variable can not be reassigned

We can manipulate values of keys but cannot overwrite the object value

      
        const obj = {'firstName':'John'};
        const obj = {'name':'Toad'}; // throws error, can reassign an const variable
        const obj.firstName = 'Toad'; //acceptable, as the key is not protected
