Default parameters

The default value for function parameters is undefined. This can be achieved with the help of default function parameters.

      
        function fullName(firstName, lastName ='John', fullName = firstName + ' ' + lastName ) {
          return fullName;
        }
        fullName('Daliya'); // return Daliya John

Template literals (Template strings)

Template literals allow embedded expressions, multi-line strings and string interpolation.

      
        let myName = 'Daliya';
        let myRole = 'Software Developer';
        console.log(`My name is ${myName} and I am a ${myRole}.`); // My name is John and I am a Software Developer
